# Trabalho PAA — Algoritmo de Corte Máximo

## Descrição

Este repositório é um projeto prático para a disciplina Projeto de Análise de Algoritmos (PAA). O foco é a implementação, experimentação e análise do algoritmo de Corte Máximo aplicado a instâncias de grafos.

## Objetivo

Fornecer uma base prática para estudar o comportamento do algoritmo de Corte Máximo, comparar variantes (exatas e heurísticas quando aplicável), coletar resultados experimentais e analisar complexidade e desempenho.

## Conteúdo

- Implementações do(s) algoritmo(s) relacionados ao Corte Máximo.
- Scripts para gerar e executar instâncias de teste.
- Jupyter notebooks e/ ou scripts de análise para plotagem de resultados e comparação de desempenho.
- Pasta `data/` para armazenar instâncias de teste e `results/` para saídas experimentais.

## Requisitos

- Python 3.8+
- Recomenda-se usar um ambiente virtual (`venv` ou similar).

Exemplo rápido de instalação das dependências (se houver um arquivo `requirements.txt`):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Estrutura do repositório

- `src/` — código-fonte principal
- `scripts/` — scripts utilitários para execução e geração de instâncias
- `data/` — instâncias de entrada
- `results/` — resultados e saídas de experimento
- `notebooks/` — análises e visualizações em Jupyter
- `README.md` — este arquivo

## Análise e relatórios

Inclua scripts ou notebooks que calculem métricas relevantes (qualidade da solução, tempo de execução, uso de memória) e que gerem gráficos comparativos para diferentes instâncias e variantes do algoritmo.
